# Student API

Simple REST API built with Java and Spring Boot.

## Features
- Get student list
- REST endpoint returning JSON
- Basic Spring Boot backend project

## Endpoint

GET /students

Example response:

[
  {"id":1,"name":"Irem","department":"Computer Engineering"},
  {"id":2,"name":"Mehmet","department":"Mechanical Engineering"},
  {"id":3,"name":"Ayse","department":"Architecture"}
]

## Technologies

- Java
- Spring Boot
- Maven
- Git & GitHub
